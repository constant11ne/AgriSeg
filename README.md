# AgriSeg

Проект посвящён семантической сегментации аномалий на мультиспектральных снимках сельскохозяйственных полей с использованием RGB, NIR и NDVI данных.

Работа выполнена в рамках ВКР на тему **«Использование адаптированных методов сегментации для анализа мультиспектральных снимков сельскохозяйственных полей»**. Основная задача проекта — исследовать, как разные способы объединения спектральных модальностей влияют на качество сегментации.

## Описание

В проекте реализован экспериментальный пайплайн для обучения и сравнения моделей семантической сегментации на датасете Agriculture Vision.

Исследуются следующие подходы:

- RGB-only и NIR-only модели;
- раннее объединение RGB и NIR каналов;
- позднее объединение предсказаний;
- mid-level fusion архитектуры;
- progressive fusion по уровням пирамиды признаков;
- использование NDVI как дополнительного спектрального признака;
- RGB+NDVI fusion;
- IBN alignment для согласования признаков разных модальностей.

Лучший результат в экспериментах был получен моделью **RGB+NDVI mid-level fusion с IBN alignment**, достигшей `mIoU = 0.5418`.

```text
src/

  models/             # модели и fusion-архитектуры
  utils/              # анализ результатов и вспомогательные функции
  train.py            # основной скрипт обучения

notebooks/
  experiments_*.py    # запуск серий экспериментов
  analyze_*.py        # анализ результатов

tools/
  run_inference.py    # инференс сохранённых моделей
````

## Основные возможности

* обучение RGB, NIR и мультимодальных моделей;
* поддержка разных стратегий fusion;
* вычисление NDVI;
* сохранение конфигураций и метрик экспериментов;
* анализ результатов по сериям запусков;
* инференс по сохранённым чекпоинтам.

## Пример запуска

```bash
python -m src.train \
  --data-root ../AgricultureVision \
  --model fpn \
  --encoder timm-efficientnet-b4 \
  --epochs 35 \
  --batch-size 16 \
  --img-size 256 256 \
  --loss soft_bce_dice
```

## Результаты

Ключевые результаты экспериментов:

| Модель                     | Best mIoU |
| -------------------------- | --------: |
| RGB-only                   |    0.4724 |
| NIR-only                   |    0.4554 |
| Early RGB+NIR              |    0.4836 |
| Progressive RGB+NIR fusion |    0.5181 |
| Late RGB+NIR fusion        |    0.4985 |
| Mid RGB+NDVI               |    0.5343 |
| Mid RGB+NDVI + IBN         |    0.5418 |

## Вывод

Эксперименты показали, что простое добавление NIR или NDVI как дополнительного входного канала не всегда улучшает качество. Наиболее эффективным оказался подход, где NDVI используется как отдельная модальность и объединяется с RGB-признаками на уровне feature maps.
